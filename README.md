# HCL Internship Management Portal

A full-stack internship management system developed for HCL internship operations.

## Features

- Student Registration & Login
- Admin Dashboard
- Attendance Management
- Performance Tracking
- Student Profiles
- AI Assistant Chatbot
- Internship Reports
- Responsive UI

## Tech Stack

Frontend:
- HTML
- CSS
- JavaScript

Backend:
- Node.js
- Express.js

Database:
- MongoDB# hcl-internship-management
